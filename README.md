# Weather-app

🌤 Weather App universal

This Weather App is a simple and responsive web application that provides real-time weather information for any city worldwide. Built using HTML, CSS, and JavaScript, it allows users to enter a city name and instantly view current temperature and weather conditions. The app uses a weather API to fetch live data and display it dynamically on the screen. It demonstrates key concepts like API integration, asynchronous JavaScript, and DOM manipulation. Designed for simplicity and ease of use, this project is ideal for beginners learning web development and building interactive, real-world applications.
